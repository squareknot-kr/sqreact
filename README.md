# sq-react

React 컴포넌트와 유틸리티 라이브러리

## 설치

```bash
pnpm add sq-react
# 또는
npm install sq-react
# 또는
yarn add sq-react
```

## 사용법

```tsx
import { ComponentName } from 'sq-react';
import { utilityFunction } from 'sq-react';

// 컴포넌트 사용
function App() {
  return <ComponentName />;
}
```

## 개발

```bash
# 의존성 설치
pnpm install

# 개발 모드
pnpm dev

# 빌드
pnpm build

# 타입 체크
pnpm type-check
```
